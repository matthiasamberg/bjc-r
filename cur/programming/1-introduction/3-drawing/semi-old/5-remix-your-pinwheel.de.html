Auf dieser Seite nutzt du deinen pinwheel-Block, um asterisk- und polygon-Blöcke zu bauen.

Dein allgemeiner Pinwheel-Block kann als Grundlage für andere, speziellere Prozeduren dienen, die jeweils eine konkrete Aufgabe übernehmen.

  1. Öffne dein U1L3-Pinwheel-Projekt.
  2. Erstelle auf Basis deines pinwheel-Blocks einen spezialisierten Block asterisk.
    1. Erzeuge einen neuen Block. Gib ihm Namen und Farbe deiner Wahl.
    2. Füge Eingaben hinzu (z. B. branches, size, backup) und nutze sie im Inneren, indem du deinen Pinwheel-Block mit passenden Werten aufrufst.
    3. Teste verschiedene Eingaben, um sicherzugehen, dass dein Block korrekt funktioniert.
  1. Schreibe analog einen Block polygon, der mithilfe von pinwheel Polygone mit gewünschter Seitenzahl und Seitenlänge zeichnet.
: Abstraktion durch Verallgemeinerung

Auf Einheit 1 Lab 2, Seite 2 hast du prozedurale Abstraktion kennengelernt: Code benennen, damit man ihn mehrfach aufrufen kann.

(AAP-)3.B.4, DAT-1.A.5

Hier siehst du eine weitere Form der Abstraktion: Erkennen, dass asterisk und polygon Varianten desselben Pinwheels sind, und daher einen allgemeinen Block pinwheel mit Eingaben zu bauen, der alle Varianten abdeckt.

(AAP-)3.B.5

Das nennt man Abstraktion durch Verallgemeinerung: Man betrachtet ähnlich aufgebauten Code und schreibt einen einzigen Block, der mithilfe von Eingaben die Unterschiede handhabt.
