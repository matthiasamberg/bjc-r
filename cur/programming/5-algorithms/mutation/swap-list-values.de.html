Wir möchten einen Block bauen, der zwei Zeilen in einer Liste vertauscht. Der Block bekommt zwei Zahlen – die Positionen der Elemente – und eine Liste als Eingabe und tauscht anschließend die Elemente an diesen Positionen.

Damit das funktioniert, müssen wir etwas tun, das wir zuvor noch nicht gebraucht haben: eine Liste als Eingabewert akzeptieren. Die dritte Eingabe list soll eine Liste sein. Klicke auf die Variablen rowX und rowY und lege fest, dass ihre Eingabetypen Zahlen sind. Danach klickst du auf list und setzt den Eingabetyp auf „List“. Das folgende Dialogfenster zeigt die Auswahl.

Nach einem Klick auf OK siehst du den Block mit zwei runden Zahleneingaben und einem Listen-Eingabefeld.

Im Block Editor können wir die Variable list dann wie eine normale Liste verwenden, z. B. indem wir der Blockvariablen temp-value das Element an Position rowX aus list zuweisen.
