In diesem Labor erstellst du Werkzeuge, um Daten zu speichern und abzurufen.

Auf dieser Seite baust du eine Einkaufsliste-App.

Manche Programmiersprachen verwenden statt Liste das Wort Array.

Viele Computeranwendungen — Kontaktlisten, Playlists, Kalender, Erinnerungen — arbeiten mit Listen von Informationen, die durchsucht, sortiert oder verändert werden. Du hast bereits mit Listen gearbeitet, als du das Gossip-Projekt angepasst hast.

AAP-1.C.1 (zweiter Satz), AAP-1.D.7
Wie sieht das in Python aus? 
["Señora", "Ms. C", "my cat", "Hannah", "Jake"]
Der Listen-Ausdruck wird als 
[Señora, Ms. C, my cat, Hannah, Jake]
geschrieben. Die Elemente stehen in der Reihenfolge, in der sie im Text erscheinen: „Señora“ hat den Index 1, „Ms. C“ den Index 2 usw.

Elemente aus einer Liste entfernen
