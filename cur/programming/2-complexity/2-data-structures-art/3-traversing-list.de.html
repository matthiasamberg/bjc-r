AAP-2.O.2

Eine Liste zu traversieren bedeutet, jedes Element der Liste anzuschauen. for each ist iterativ, also wiederholend, genau wie for, das ebenfalls eine Liste traversieren kann. Im Gegensatz zu for arbeitet for each jedoch ohne Indexzahlen.
