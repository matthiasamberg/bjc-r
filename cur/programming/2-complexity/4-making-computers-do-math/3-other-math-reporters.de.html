Der Block bekommt zwei Eingaben: eine Liste und eine Operation mit zwei Eingabefeldern. Er liefert das Ergebnis, das entsteht, wenn du alle Elemente der Liste mit dieser Operation kombinierst. Der Ausdruck unten berechnet zum Beispiel 0 + 1 + 2.

Du entscheidest, welche Operation angewendet wird; combine führt sie dann auf alle Listenelemente aus und liefert das Ergebnis.

Beachte, dass die Funktion zum Kombinieren immer zwei leere Eingabefelder hat. Beim Block keep reicht ein Platzhalter, bei combine sind zwei Pflicht.

combine ist eine Higher-Order-Funktion; sie nimmt selbst eine Funktion als Eingabe. Eine weitere Higher-Order-Funktion kennst du bereits: keep (siehe Einheit 2, Labor 3).

Warum so wenige?

Die Funktion muss assoziativ sein, d. h. es darf egal sein, wie die Elemente gruppiert werden. Addition ist assoziativ: (7 + 8) + 1 ist dasselbe wie 7 + (8 + 1). Subtraktion ist es nicht: (7 − 8) − 1 liefert ein anderes Ergebnis als 7 − (8 − 1). Daher wäre ein Ausdruck wie zweideutig.

  1. Verwende combine, um einen Block sum zu erstellen, der alle Elemente einer Liste addiert. Teste ihn mit mehreren Listen (einschließlich leerer Listen).
  2. Erstelle auf dieselbe Weise einen Block product, der die Elemente multipliziert. Was soll er zurückgeben, wenn die Liste leer ist?
  3. Baue einen Block all true?, der über eine Liste Boolescher Werte entscheidet, ob alle Einträge true sind, und einen Block any true?, der prüft, ob mindestens ein Eintrag true ist.
