Elemente mit keep aus einer Liste behalten

Auf dieser Seite verwendest du Prädikate gemeinsam mit dem Block keep, um nur die Listeneinträge zu behalten, die bestimmte Eigenschaften erfüllen.

Prädikate kannst du mit Bedingungsblöcken kombinieren, um zu entscheiden, wann etwas passieren soll; mit keep legst du fest, welche Einträge erhalten bleiben. Der Block bekommt eine Liste und ein Prädikat als Eingabe und liefert eine neue Liste, die nur die Elemente enthält, für die das Prädikat true ergibt.

Der folgende Ausdruck findet beispielsweise alle Wörter aus der Liste words list, deren erster Buchstabe v ist. Das leere Eingabefeld ist der Platzhalter für das jeweilige Listenelement, über das entschieden wird.

: String und Index
  • AAP-1.C.4
    Ein String ist eine geordnete Folge von Zeichen.
  • AAP-1.C.3 (strings)
    Den Begriff Index kennst du bereits als Positionsnummer eines Elements in einer Liste. Derselbe Begriff bezeichnet auch die Position eines Zeichens in einem String (z. B. eines Buchstabens in einem Wort). Der erste Eingang von letter ist also der Index.

Du schreibst das Prädikat, das die Prüfung übernimmt, und keep wendet dieses Prädikat auf jedes Listenelement an und liefert die Elemente zurück, für die dein Prädikat true meldet.

Das leere Eingabefeld im Prädikat ist notwendig; hier wird jedes Element eingesetzt, während es geprüft wird.
