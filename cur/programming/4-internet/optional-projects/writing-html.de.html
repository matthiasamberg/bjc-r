HTML schreiben (Teacher’s Choice)

Snap! ist eine universelle Programmiersprache für unterschiedliche Algorithmen. HTML dagegen ist eine spezialisierte Sprache – sie dient ausschließlich zum Gestalten von Webseiten. HTML steuert Layout und Stil von Text/Bildern, erlaubt aber keine Algorithmen wie Snap!. Dafür nutzen Web-Entwickler Sprachen wie JavaScript.

Jede HTML-Seite besitzt ein Grundgerüst. Browser nutzen HTTP, um daraus die sichtbare Seite zu machen. Die meisten Tags (z. B. <head>) haben ein schließendes Tag (</head>), zwischen denen der Inhalt steht. Farben/Einrückungen dienen nur der Lesbarkeit.

<!DOCTYPE html>
<html>
	<head>
		<title>Titel hier</title>
	</head>
	<body>
		Inhalt hier.
	</body>
</html>
  1. Experimentiere mit diesem HTML-Beispiel. Schreibe eigenen Code zwischen <body> und </body>. Ergänze z. B. Überschriften oder Absätze.

    • Absätze: <p></p>.
