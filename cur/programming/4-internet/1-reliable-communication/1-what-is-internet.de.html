    CSN-1.B.1 (minus "open"), CSN-1.D.1, CSN-1.D.2, CSN-1.D.3
    : Internet, Computernetzwerk, Computersystem, Computergerät und World Wide Web
    • Das Internet ist ein Computernetzwerk, das offene Protokolle verwendet, um Kommunikation zu standardisieren. Um auf das Internet zugreifen zu können, brauchst du ein Computergerät, das mit dem Internet verbunden ist.
        CSN-1.A.1, CSN-1.A.2, CSN-1.A.3, CSN-1.A.4, CSN-1.B.1, CSN-1.B.2, CSN-1.D.1, CSN-1.D.3
      • Ein Computernetzwerk ist ein verbundenes Computersystem, das Daten senden oder empfangen kann.
      • Ein Computersystem ist eine Gruppe von Computergeräten und Programmen, die gemeinsam ein Ziel verfolgen.
      • Ein Computergerät ist ein physisches Objekt, das ein Programm ausführen kann, etwa Computer, Tablets, Mobiltelefone oder intelligente Sensoren.
    • Das World Wide Web ist ein System verknüpfter Seiten, Programme und Dateien, das das Internet nutzt.

    Viele Menschen verwenden „Internet“ und „World Wide Web“ synonym, aber das stimmt nicht. Das Internet ist ein Netzwerk unabhängiger, aber miteinander verbundener Computergeräte auf der ganzen Welt. Das World Wide Web ist die Sammlung verknüpfter Webseiten, die du dir in einem Browser ansehen kannst, indem du eine Adresse wie https://snap.berkeley.edu/snap/snap.html eingibst. Die meisten Webseiten sind in HTML (HyperText Markup Language) geschrieben und werden von deinem Browser mit HTTP (HyperText Transfer Protocol) interpretiert.

    Das Internet ist allgemeiner als das World Wide Web. Es unterstützt auch E-Mails, mobile Apps, Textnachrichten (SMS oder Short Message Service), Dateitransfers und viele weitere Formen der computerbasierten Kommunikation.

    Wie funktioniert das Internet?

    Das Internet ist ein riesiges Netzwerk von Computern, das Kommunikation rund um den Globus ermöglicht. Es funktioniert, weil es fehlertolerant konstruiert ist (es kann weiterarbeiten, auch wenn Teile des Netzwerks ausfallen) und weil es Protokolle (eine Art Abstraktion) für das Routing und die Übertragung von Daten verwendet:

    • Das Internet ist fehlertolerant, weil es ein redundantes Netzwerk ist. Zwischen den physischen Verbindungen existieren mehrere Pfade, die Redundanz schaffen. Selbst wenn ein Pfad nicht verfügbar ist, gibt es noch einen anderen Weg, um eine Nachricht vom Absender zum Empfänger zu bringen (wie rechts dargestellt). Die Software in den Verbindungspunkten weiß, wie sie Daten umleiten kann, falls eine Verbindung ausfällt.
    • Daten werden über offene Protokolle übertragen. Protokolle vereinheitlichen die Kommunikation, sodass alle Daten nach denselben Regeln gesendet und empfangen werden. Diese Protokolle sind offen (frei verfügbar), damit alle Systeme entwickelt werden können, die sich mit dem Internet verbinden.

    Das Internet ist jedoch nicht einfach nur ein Netzwerk aus Computern. Es ist ein Netzwerk von Netzwerken. Die Verbindungspunkte zwischen Netzwerken nennt man Router. Das sind Netzwerkgeräte, die den Datenverkehr zwischen Teilnetzen im Internet weiterleiten. Die Auswertung der Informationen findet erst auf dem Zielrechner statt.

    : Router und ISP
    • Ein Router ist ein Computer, der Informationen von einem Netzwerk in ein anderes weiterleitet.

      • Dein Computer verbindet sich wahrscheinlich über einen Router irgendwo in deinem Zuhause mit deinem ISP.

    • ISPs (Internet Service Provider) sind Unternehmen, die Haushalten und Institutionen Internetzugang verkaufen.

      • Die Computer, die mit dem Internet verbunden sind, und die Verbindungen zwischen ihnen gehören keiner einzelnen Organisation. Unterschiedliche ISPs versorgen verschiedene Gemeinschaften mit dem Internet. Und innerhalb großer Organisationen (zum Beispiel Universitäten) stellt die Organisation selbst die Internetverbindung bereit.

    Was ist die Cloud?

    : Bandbreite und die Cloud
      CSN-1.A.7, CSN-1.A.8
    • Bandbreite ist die maximale Datenmenge, die innerhalb eines festen Zeitraums gesendet werden kann (bei digitalen Daten wird sie in Bits pro Sekunde gemessen). Eine höhere Bandbreite ist besser, weil die Daten schneller übertragen werden.
    • Daten in der Cloud zu speichern bedeutet, sie irgendwo im Internet abzulegen, ohne dass du den genauen Ort kennst. Der Cloud-Anbieter (Google, Dropbox, Amazon, Snap! usw.) verwaltet den tatsächlichen Speicherort.

    In den frühen Tagen des Internets war die Bandbreite gering, sodass Menschen keine Videos versenden konnten (nur Text und kleine Bilder). Damals waren Computer groß und teuer, also ging man zu einem Rechenzentrum, um Berechnungen durchzuführen. Als die Technik kleiner und effizienter wurde, verloren diese Zentren an Bedeutung, weil Menschen Berechnungen auf persönlichen Computergeräten ausführen konnten.

    In jüngster Zeit benötigen bestimmte Arten von Berechnungen (etwa Websuchen und Spracherkennung) jedoch wieder viel Rechenleistung, weshalb diese Aufgaben an riesige „Computerfarmen“ ausgelagert werden, in denen Zehntausende Computer gemeinsam an einem Problem arbeiten. Diese Computerfarmen werden zusammenfassend als die Cloud bezeichnet. Du hast die Cloud in diesem Kurs bereits genutzt: Alle deine Snap!-Projekte werden in der Cloud gespeichert. Du benutzt zwar weiterhin deinen Computer vor Ort, doch einige Programme laufen tatsächlich in der Cloud.

    Wenn du deine Snap!-Projekte in deinem Konto speicherst, liegen sie nicht auf deinem lokalen Computer, sondern in der Cloud. Deshalb musst du dich anmelden, um darauf zuzugreifen. Ähnlich verhält es sich mit Google Drive oder Dropbox: Auch dort werden alle Dateien in der Cloud gespeichert. Und wenn du einen webbasierten E-Mail-Dienst (z. B. Gmail oder Yahoo) nutzt, werden deine E-Mails ebenfalls in der Cloud abgelegt.
