Eisschalen zählen

In diesem Labor untersuchst du ein klassisches Problem der Informatik: Teilmengen.

Auf dieser Seite startest du mit einem vertrauten Szenario – der Auswahl von Eissorten.

: Teilmengen

Eine Teilmengen ist eine Auswahl von Elementen aus einer Menge – sie kann leer sein, alle Elemente enthalten oder irgendetwas dazwischen.

Woher stammt dieses Bild? Braucht es eine Quellenangabe? --MF, 22.04.2022
Viele Aufgaben drehen sich um Teilmengen, z. B.:
  • In einer Klasse sind 20 Schüler*innen, du brauchst ein Vierer-Komitee. Wie viele Kombinationen sind möglich?
  • Ein Simplex-Schloss besitzt fünf Tasten; jede darf höchstens einmal benutzt werden, mehrere Tasten dürfen gleichzeitig gedrückt werden. Wie viele Kombinationen gibt es?
  • Ein Laden bietet acht Eissorten. Wie viele Schalen-Kombinationen gibt es für eine vorgegebene Anzahl an Kugeln?

Für den Anfang gilt: Eine Sorte pro Kugel, Reihenfolge egal (Vanille+Schoko = Schoko+Vanille).
  1. Wie viele Zwei-Kugel-Schalen gibt es? Finde eine systematische Auflistung.
  2. Überlege, wie sich das Problem rekursiv beschreiben lässt.
  3. „U8L3-Subsets“
  4. Vielleicht brauchst du den Block append.
    Baue einen Block, der bei einer Liste verfügbarer Sorten alle Zwei-Kugel-Schalen zurückgibt. Da eine Schale als Liste von zwei Sorten dargestellt wird, erhältst du eine Liste von Listen.
Der Zweier-Block hilft dir beim Dreier-Block.
  1. Erstelle einen Block, der jede Drei-Kugel-Schale auflistet. Achte darauf, Duplikate zu vermeiden (Reihenfolge egal!).
  2. Eine Schale enthält Schokolade – oder eben nicht.
    Verallgemeinere auf einen Block, der alle n-Kugel-Schalen liefert (zusätzliches Eingabefeld für die Kugelanzahl). Hier brauchst du Rekursion!
  1. Schreibe eine Prozedur, die bei einer Sortenliste alle möglichen Schalen zurückgibt – unabhängig von der Kugelanzahl.
  2. Hebe die Regel auf, dass jede Kugel eine andere Sorte haben muss. „Schokolade, Schokolade, Schokolade“ ist jetzt erlaubt – beachte, dass es dann nicht mehr um Teilmengen im strengen Sinn geht, denn Mengen enthalten kein Element doppelt.
