So könnte eine Binärumrechnung für nichtnegative Ganzzahlen aussehen:

Wenn sich eine Zahl mit nur einer Ziffer darstellen lässt, ist das der Basisfall. Im Binärsystem gibt es dafür nur 0 und 1, daher prüfen wir, ob number < 2 ist.

Überraschend: Wir nutzen keinen arithmetischen Kombinierer, denn das Ergebnis soll ein Ziffernstring sein – daher verwenden wir join.
Im rekursiven Fall ist das rechte Bit der Rest bei Division durch 2. Die übrigen Bits stammen aus einem rekursiven Aufruf auf den Quotienten. Der Kombinierer ist join, weil wir Ziffern „zusammenkleben“.
