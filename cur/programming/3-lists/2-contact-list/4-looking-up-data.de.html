  1. Öffne dein Projekt U3L2-ContactList.
  2. Schreibe einen Block , der einen Text entgegennimmt und entweder eine Liste aller Kontakte zurückgibt, deren Name diesen Text enthält, oder eine leere Liste, falls kein Treffer vorhanden ist.
    Nützliche Blöcke:
    1. (liefert true, wenn der erste String den zweiten enthält)
    2. (behält nur Elemente, für die das Prädikat wahr ist)
    3. oder dein Selektor für den Namen
  3. Teste den Block mit unterschiedlichen Suchtexten (voller Name, nur Vorname, nur Nachname).
  4. Erstelle den Button „Kontakt suchen“, der den Suchtext abfragt, find contacts nutzt und das Ergebnis auf der Bühne anzeigt.
  6. Debugge und stelle sicher, dass auch mehrere Treffer korrekt dargestellt werden.
